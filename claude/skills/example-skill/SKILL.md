# Example Skill

このスキルは〇〇するための標準手順をまとめたもの。

## いつ使うか

- ユーザーが「△△したい」と言ったとき
- 「□□して」と頼まれたとき
- 〇〇に関する作業を行うとき

## 前提条件

- `node` v20 以上がインストールされていること
- 環境変数 `EXAMPLE_API_KEY` が設定されていること

## 手順

### Step 1: 入力を確認する

ユーザーから以下の情報を受け取る:
- 対象ファイル / URL / ID
- オプション設定

不足していれば質問して補完する。

### Step 2: 実行する

```bash
node scripts/run-example.js --target <input>
```

### Step 3: 結果を検証する

- 出力ファイルが生成されたか確認
- ログにエラーがないか確認
- 必要に応じてテストを実行

### Step 4: ユーザーに報告する

以下のフォーマットで結果を提示:

```
## 実行結果
- 入力: ...
- 出力: ...
- 所要時間: ...

## 次の一手
1. **〇〇する** — ...
2. △△を確認する — ...
```

## エラーハンドリング

| エラー | 原因 | 対処 |
|---|---|---|
| `EACCES` | 権限不足 | `chmod +x scripts/run-example.js` |
| `ECONNREFUSED` | サーバ未起動 | `npm run dev` で起動 |
| `Invalid token` | API key 不正 | `.env` を確認 |
